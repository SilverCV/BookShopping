### 简介

一个简单的javaweb项目。

使用了jsp,servlet,数据库使用Mysql，前端效果部分使用了bootstrap和jquery。

### 实现了（但仍有bug）

* 前台注册登录
* 搜索图书（按作者，书名）
* 后台管理（管理员信息，用户信息，书籍类别，书籍信息）

### 未实现以及bug

太多了...
并不能完全算是一个购书网站吧...
